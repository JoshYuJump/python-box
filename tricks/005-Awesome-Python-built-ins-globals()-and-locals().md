# "globals()" returns a dict
# with all global variables
# in the current scope:

>>> globals()
{...}

# "locals()" does the same
# but for all local variables
# in the current scope:

>>> locals()
{...}

globas()方法返回了一个字典，
这个字典中包含了在当前作用域，
可以获取到的所有全局变量的值。

locals()也是相同的含义，
但是表示的是，在当前作用域，
可以获取的局部变量
（一般的局部变量，只作用于当前作用域）
