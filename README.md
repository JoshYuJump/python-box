# python-box 介绍 

本项目旨在收集并翻译 Python 的开发技巧的有价值的内容。
收集到的技巧会发布在 `issues` 里面，有兴趣的朋友可以在 `issues` 通过评论的方式领取翻译任务，并提交翻译后的 `Pull Request`。
